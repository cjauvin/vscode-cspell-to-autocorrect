# cspell-to-autocorrect

A VS Code extension that adds cSpell corrections to an Auto Correct dictionary.

See: https://cjauvin.github.io/posts/vscode-vibe-coding/

## Features

When you correct a spelling error using cSpell, this extension can automatically add the correction to an autocorrect dictionary for future use.

## Usage

Use the command `Apply cSpell correction and add it to Auto Correct dictionary` to apply a spelling correction and save it.
